generate_expert_demonstrations_feeder.expert_rollouts(env, episodes, _expert=True)¶

This function performs roll-outs on the Open-DSS environment to obtain trajectory samples

Parameters

  • env (gym.Env) – The OpenDSS RL environment

  • episodes (int) – number of episodes the rollout to be performed.

  • _expert (bool) – if set true then generate expert trajectories or random trajectories

Returns

average episode length and trajectories/rollouts

Return type

int, Trajectories
